What data is collected?

Uchi itself does not collect any data (as the source code shows), however:

  • if using a third-party instance, there may be some form of server-side logging (which is usually harmless); in this case, to get information, you must contact the instance administrator
  • your IP address is shared with OkSurf in order to provide the news headlines

Cookies

Uchi uses the Web Storage API as a caching mechanism in order to improve performance. Local storage only works on your device: you can delete this data in your browser settings.
